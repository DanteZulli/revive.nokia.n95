RM-160 Phone Data Package NAM v4.0
========================================
Creation date: Fri May 23 16:30:07 2008

Important notes
-------------------------------------------
! Please remove any previously installed 
! data package before installing this one.
-------------------------------------------
*NOTE	We recommend the removal of previous data package before each new install.
        Otherwise some old images may be are shown in SW update 
        (for those who don't have new versions in this distribution)

*NOTE	Phoenix must be closed when installing this Data Package.
         
*NOTE	Phoenix Internal Service Software A 2007.04.6.170 (or newer) and 
	Prommer (FPS-10) Flash Update package  07.04.013 needed.

*NOTE	This is DP2.0 data package. Use Firmware Update UI for flashing.
        
*NOTE	Please read the flashing guide related to this release.


Target HW
-----------------------------------------
B4 or newer


Software version
-----------------------------------------
APE SW: 20.2.011 
 - Localised production (Prd) images
CMT SW: 06wk41v29.1
